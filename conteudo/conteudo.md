# Diferenças básicas entre C++ e Python

# Funcoes
Obs: Todo arquivo c++ deve ter uma função chamada main() definida nele, seu programa irá começar a rodar a partir dela.

Assim como em python, funções são agrupamentos de código destinados a rodar em sequência para algum fim específico ao chamarmos a função no código. Em c++, nós também possuimos funções embutidas. Então, os dois tipos de funções são as embutidas (already-writtens) e que nós escrevemos (user-defined).

# Arrays
Um array é um tipo de dado container que armazena uma sequencia de valores continuamente (o que significa que não há espaços vazios na memória entre um valor do nosso array e outro). Arrays nos permitem um acesso rápido e direto a qualquer elemento dele.
